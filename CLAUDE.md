# Hamilton Co-Ed Classic - Project Specs

## League Configuration
- **Teams:** 10
- **Game Cap:** 21 games per team max.
- **Location:** Turner Park.
- **Diamond Constraints:** - Available: Diamonds 2, 3, 4, 5, 6.
  - No Lights: Diamonds 2 & 3 (Games must end by dusk).
  - Invalid: Diamond 1.

## Technical Stack
- **Backend:** Google Apps Script (Code.gs)
- **Frontend:** HTML Service (index.html) using Bootstrap/Tailwind.
- **Data Source:** Google Sheets.

## Coding Standards
- **Function Naming:** camelCase.
- **Full File Output:** Every response containing a code change must include the entire file.
- **Verification:** Before providing a new schedule, verify that no team exceeds 21 games.
